# Day 19 — Prime Number Check

Is it prime? ��

Write a function called `isPrime` that takes a number and returns `true` if it's prime, `false` otherwise. A prime number is only divisible by 1 and itself.

**Example:**
```javascript
isPrime(7) → true
isPrime(10) → false
isPrime(13) → true
isPrime(1) → false
```

**Your task:**
Create the function. If `isPrime(17)` is true, the code is "PRIME17", otherwise "NOTPRIME17".

**Hint:** Check if any number from 2 to √n divides it evenly!
