# Day 18 — Title Case

Format text properly! ✍️

Write a function called `toTitleCase` that takes a string and returns it with the first letter of each word capitalized.

**Example:**
```javascript
toTitleCase("hello world") → "Hello World"
toTitleCase("the quick fox") → "The Quick Fox"
toTitleCase("code") → "Code"
```

**Your task:**
Create the function and format "merry christmas". The unlock code is this result (including the space).

**Hint:** Split by spaces, capitalize first letter of each word, then join!
